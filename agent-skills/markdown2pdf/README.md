# md-to-pdf: Gemini CLI Skill

A specialized skill for Gemini CLI that consolidates multiple Markdown files into a professional, high-fidelity PDF using Pandoc and Chromium. This skill is fully compliant with the **agents.md** specification.

## Prerequisites

To use this skill, ensure the following are installed on your system:
- **Pandoc**: `sudo apt install pandoc` (or equivalent)
- **Chromium**: `sudo apt install chromium-browser` (or `chromium`)
- **Node.js**: v18 or higher

## Installation

You can install this skill either for the current workspace or globally for all projects.

### Workspace Level (Recommended for project-specific docs)
```bash
gemini skills install md-to-pdf.skill --scope workspace
```

### User Level (Global)
```bash
gemini skills install md-to-pdf.skill --scope user
```

### Activate the Skill
After installation, you **must** reload your skills in your interactive session:
```bash
/skills reload
```

## Usage

Once installed, you can ask Gemini CLI to:
- "Consolidate all markdown files in the docs folder into a design document."
- "Create a PDF from SPEC.md and ROADMAP.md."
- "Generate a professional report from these notes."

### Manual Execution (via script)
The skill includes a standalone script that can be run directly:
```bash
node path/to/skill/scripts/md2pdf.cjs <output.pdf> <input1.md> <input2.md> ...
```

## Internal Structure
- `SKILL.md`: Core AI instructions and workflow.
- `AGENTS.md`: Machine-readable specification for AI agents.
- `scripts/md2pdf.cjs`: The conversion engine.
