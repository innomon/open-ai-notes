## [Anthropic Just Changed How Agents Call Tools. I Stole It for My Qwen3.5 Agent.](https://www.youtube.com/watch?v=R7OCrqyGMeY)

### Description Summary
The video discusses a significant shift in how AI agents interact with tools, moving from traditional JSON-based tool calling to **programmatic tool calling** (also referred to as "Code Mode" or "Advanced Tool Use").

Key points:
- **The Shift:** Instead of outputting a structured JSON schema, the agent now writes and executes code to orchestrate multiple tools.
- **Efficiency:** Can reduce token usage by 30% to 80% by not passing intermediate results back into the context window.
- **Implementation:** Demonstration of adapting "Anthropic-style" programmatic approaches for a **Qwen 3.5** agent.
- **Web MCP:** Touches on the new "Web MCP" standard (collaboration between Google, Anthropic, and W3C) for website tool understanding.
